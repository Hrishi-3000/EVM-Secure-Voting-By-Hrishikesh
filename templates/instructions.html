How to use the EVM Voting System
  1. Admin Panel: Add candidates and reset the system.
    • Login with admin password (default: admin123)
    • Add candidates by entering comma-separated names
    • Reset will clear all voting data
  2. Voter Panel: Enter your phone number and vote for your preferred candidate.
    • Each phone number can vote only once
    • Select your preferred candidate
  3. Results Panel: View live voting results.
    • See vote counts for each candidate
    • View pie chart visualization
    • Export results to CSV or JSON
